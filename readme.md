Moss Running Game
=================

Simple running game featuring IT Crowd's Maurice Moss.

I'm using [Blender](http://www.blender.org/) and [Unity3D](http://unity3d.com/).
The original character design is from [CubeeCraft](http://www.cubeecraft.com/).
